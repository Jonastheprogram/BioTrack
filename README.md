# 📲 BioTrack

App Android em Kotlin para Monitoramento de Biodiversidade

## 🚀 Funcionalidades

- Cadastro de usuário
- Registro de Avistamentos
- Mapa de registro e avistamentos
- Coleta de dados em tempo real
